# 1.1 异或运算

🚩 任何数字异或它自己，都为0!

问题：有一堆数字，如果除了一个数字外，其他数字都出现了两次，找出出现一次的数字。

nums := []int{1, 5, 1, 6, 5, 3, 6}
i := 0
for _, v := range nums {
  i ^= v
}
fmt.Println(i)
