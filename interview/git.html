# Git 相关

# 游离分支

Git中的游离分支（Detached HEAD）是指切换到某个提交时，HEAD指针指向的不是一个分支，而是直接指向该提交。通常有以下几种情况：

git chekcout <commit>
git checkout tag_name
  • 查看某个提交的内容
  • 切换到某个 Tag 或特定的提交
  • 在没有创建分支的情况下进行提交

HEAD处于游离状态时，任何新的提交都不会被分支所记录。

# git log 和 git reflog

  • git log 查看提交历史，包括所有分支和标签的提交记录。
  • git reflog 查看所有操作记录，包括已经被删除的提交。
